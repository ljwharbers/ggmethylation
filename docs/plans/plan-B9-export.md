# Plan B9: Export Parsed Data (`write_methylation()`)

**Goal:** Add a `write_methylation()` function that saves a `methylation_data` object's `$reads` and `$sites` data frames to disk as TSV or BED files, enabling downstream analysis outside R.

**Affected files:**
- `R/write_methylation.R` — new file
- `NAMESPACE` — add `export(write_methylation)` via `devtools::document()`
- `man/write_methylation.Rd` — generated by roxygen2

---

## New Function Signature

```r
write_methylation(
  data,
  prefix,
  format = c("tsv", "bed"),
  reads  = TRUE,
  sites  = TRUE,
  gzip   = FALSE,
  ...
)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `data` | `methylation_data` | — | Object returned by `read_methylation()`. |
| `prefix` | character(1) | — | File path stem, e.g. `"output/sample"`. Suffixes `_reads.tsv`, `_sites.tsv`/`_sites.bed` are appended. |
| `format` | character(1) | `"tsv"` | Output format for `$sites`: `"tsv"` (full columns) or `"bed"` (6-column). `$reads` is always TSV. |
| `reads` | logical(1) | `TRUE` | Whether to write the `$reads` data frame. |
| `sites` | logical(1) | `TRUE` | Whether to write the `$sites` data frame. |
| `gzip` | logical(1) | `FALSE` | If `TRUE`, compress output via `gzfile()`, appending `.gz`. |
| `...` | — | — | Reserved; currently unused. |

---

## Output File Specification

### `$reads` — always TSV

File: `{prefix}_reads.tsv[.gz]`

Columns: `read_name`, `start`, `end`, `strand`, `group` (only when `!is.null(data$group_tag)`), `mean_mod_prob`.

`mean_mod_prob` computed inline:
```r
mean_per_read <- tapply(data$sites$mod_prob, data$sites$read_name, mean, na.rm = TRUE)
data$reads$mean_mod_prob <- as.numeric(mean_per_read[data$reads$read_name])
```
Reads with no sites get `NA`.

### `$sites` — TSV

File: `{prefix}_sites.tsv[.gz]`

Columns: `position`, `mod_prob`, `read_name`, `mod_code`, `group` (if present). Straight column dump of `data$sites`.

### `$sites` — BED (6-column)

File: `{prefix}_sites.bed[.gz]`

| Column | BED field | Derivation |
|---|---|---|
| 1 | chrom | `as.character(seqnames(data$region))` |
| 2 | chromStart | `position - 1L` (0-based BED convention) |
| 3 | chromEnd | `position` (1-bp feature) |
| 4 | name | `read_name` |
| 5 | score | `mod_prob` (float in [0, 1]) |
| 6 | strand | joined from `data$reads$strand` on `read_name` |

No header line.

### Step 2 — Input validation

```r
if (!inherits(data, "methylation_data"))
  stop("'data' must be a methylation_data object.", call. = FALSE)
if (!is.character(prefix) || length(prefix) != 1L || nchar(prefix) == 0L)
  stop("'prefix' must be a non-empty character string.", call. = FALSE)

if (!reads && !sites) {
  warning("Nothing to write: both 'reads' and 'sites' are FALSE.")
  return(invisible(data))
}
```

### Step 3 — Create output directory if needed

```r
out_dir <- dirname(prefix)
if (!is.na(out_dir) && out_dir != "." && !dir.exists(out_dir))
  dir.create(out_dir, recursive = TRUE)
```

### Step 4 — Compute `mean_mod_prob`

```r
if (nrow(data$sites) > 0L) {
  mp <- tapply(data$sites$mod_prob, data$sites$read_name, mean, na.rm = TRUE)
  data$reads$mean_mod_prob <- as.numeric(mp[data$reads$read_name])
} else {
  data$reads$mean_mod_prob <- NA_real_
}
```

### Step 5 — Write `$reads` (when `reads = TRUE`)

```r
read_cols <- c("read_name", "start", "end", "strand")
if (!is.null(data$group_tag) && "group" %in% names(data$reads))
  read_cols <- c(read_cols, "group")
read_cols <- c(read_cols, "mean_mod_prob")
.write_tsv_maybe_gz(data$reads[, read_cols, drop = FALSE],
                    paste0(prefix, "_reads.tsv"), gzip)
```

### Step 6 — Write `$sites` (when `sites = TRUE`)

TSV branch:
```r
site_cols <- c("position", "mod_prob", "read_name", "mod_code")
if ("group" %in% names(data$sites)) site_cols <- c(site_cols, "group")
.write_tsv_maybe_gz(data$sites[, site_cols, drop = FALSE],
                    paste0(prefix, "_sites.tsv"), gzip)
```

BED branch:
```r
chrom <- as.character(GenomicRanges::seqnames(data$region))
strand_map <- data$reads$strand
names(strand_map) <- data$reads$read_name
bed <- data.frame(
  chrom      = chrom,
  chromStart = data$sites$position - 1L,
  chromEnd   = data$sites$position,
  name       = data$sites$read_name,
  score      = data$sites$mod_prob,
  strand     = strand_map[data$sites$read_name],
  stringsAsFactors = FALSE
)
.write_bed_maybe_gz(bed, paste0(prefix, "_sites.bed"), gzip)
```

### Step 7 — Internal helpers (unexported)

```r
.write_tsv_maybe_gz <- function(df, path, gzip) {
  if (gzip) path <- paste0(path, ".gz")
  con <- if (gzip) gzfile(path, "w") else file(path, "w")
  on.exit(close(con))
  write.table(df, con, sep = "\t", quote = FALSE,
              row.names = FALSE, col.names = TRUE)
  invisible(path)
}

.write_bed_maybe_gz <- function(df, path, gzip) {
  if (gzip) path <- paste0(path, ".gz")
  con <- if (gzip) gzfile(path, "w") else file(path, "w")
  on.exit(close(con))
  write.table(df, con, sep = "\t", quote = FALSE,
              row.names = FALSE, col.names = FALSE)
  invisible(path)
}
```

## Edge cases

| Scenario | Handling |
|---|---|
| Empty `$reads` or `$sites` | Write zero-row file with header (TSV) or no rows (BED). No error. |
| `$sites` has no `group` column even though `$reads` does | Guard each independently: `"group" %in% names(data$reads)` and `"group" %in% names(data$sites)`. |
| `prefix` points to non-existent directory | Auto-create with `dir.create(..., recursive = TRUE)`. |
| BED score field | Written as float [0, 1] (mod_prob as-is). Note: strict BED spec requires integer 0–1000, but most tools accept floats. |
| `mod_prob` is `NA` for some sites | TSV: write `NA`. BED: write `NA` in score column. |
| `reads = FALSE, sites = FALSE` | Early warning + `invisible(data)`. |

## Testing approach

Create `tests/testthat/test-write_methylation.R`. Use a `make_mock_md()` helper (no BAM required):

```r
make_mock_md <- function(grouped = FALSE) {
  reads <- data.frame(read_name = c("r1","r2"), start = c(100L,200L),
                      end = c(300L,400L), strand = c("+","-"),
                      stringsAsFactors = FALSE)
  sites <- data.frame(position = c(150L,250L), mod_prob = c(0.9,0.1),
                      read_name = c("r1","r2"), mod_code = c("m","m"),
                      stringsAsFactors = FALSE)
  if (grouped) { reads$group <- c("H1","H2"); sites$group <- c("H1","H2") }
  structure(list(reads=reads, sites=sites,
                 region=GenomicRanges::GRanges("chr1", IRanges::IRanges(100,400)),
                 mod_code="m", group_tag=if(grouped)"HP" else NULL,
                 snv_position=NULL), class="methylation_data")
}
```

Key test cases:
1. TSV round-trip: write, read back, check `ncol`, `nrow`, column names, `mean_mod_prob` values.
2. BED output: no header, 6 columns, `chromStart == position - 1`, score equals `mod_prob` (float in [0, 1]).
3. Grouped: `group` column present in TSV `_reads` and `_sites` files.
4. Empty data: zero-row files written without error.
5. `reads = FALSE`: only `_sites.*` created.
6. `sites = FALSE`: only `_reads.tsv` created.
7. `gzip = TRUE`: file names end `.gz`; readable via `read.table(gzfile(...))`.
8. Non-`methylation_data` input: `expect_error()`.
9. Both `reads = FALSE, sites = FALSE`: `expect_warning()`, no files written.
